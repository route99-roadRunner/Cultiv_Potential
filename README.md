# PDF to Long PNG Converter

PDF 파일을 길게 이어진 하나의 PNG 이미지로 변환해주는 프로그램입니다.

## 사용 방법

### 1. 실행 파일 사용하기 (가장 간편함)
`dist` 폴더 안에 있는 **`pdf_to_long_png.exe`** 파일을 실행하세요.

1. `pdf_to_long_png.exe`를 실행합니다.
2. 변환하고 싶은 PDF 파일을 프로그램 아이콘이나 실행된 창으로 **드래그 앤 드롭** 하세요.
3. 실행 시 **메시지 박스**로 성공 또는 실패 알림이 뜹니다.
4. 변환된 이미지는 **원본 PDF 파일과 같은 위치**에 `_long.png`가 붙어 저장됩니다.

*(참고: 이 프로그램은 윈도우 모드로 실행되어 콘솔 창이 뜨지 않습니다.)*

### 2. Python 스크립트로 실행하기
Python 개발 환경이 있다면 소스 코드를 직접 실행할 수 있습니다.

```bash
# 의존성 설치
pip install -r requirements.txt

# 실행
python pdf_to_long_png.py [변환할_PDF_파일.pdf]
```

## 주요 기능
- **고해상도 변환**: PDF 페이지를 2배 확대(Zoom x2.0)하여 렌더링하므로 글자가 깨지지 않고 선명합니다.
- **자동 병합**: 모든 페이지를 세로로 이어 붙여 하나의 긴 이미지로 만듭니다.
- **자동 저장**: 별도로 저장 경로를 지정하지 않으면 원본 파일명 뒤에 `_long.png`를 붙여 자동 저장합니다.
